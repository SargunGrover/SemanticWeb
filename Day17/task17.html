Using Google Search Console

  1. Add Your Domain: Input your domain into Google Search Console.
  2. Verification: Download the verification file (HTML file or DNS record) provided by Google.
  3. Upload File: Upload this file to your website's repository or update your DNS settings.
  4. Verification Check: Google checks the file or DNS record to verify ownership of the domain.
  5. Indexing: After verification, it may take a few days for Google to index your site.
  6. Monitoring: Once indexed, you can view impressions, clicks, and performance data in Search Console.
